Calculator

Dec. 2022

Calculator is a one page project. This project presents a basic, non scientific calculator, to handle simple computations. Simple does not mean easy: a lot of user input validations had to be done.

Part of a Front-End training, the purpose of this project is to handle multiple exceptions that can occur. Focus is put mainly on Javascript and the structure of the code. This project also offers a correct user experience, supporting mouse and keyboard inputs.

Live Code

What have I learned ?

Deeper understanding of Javascript

Focus on user experience

Handle multiple exceptions, validation of user inputs

Code structure, files separated into modules to keep architecture optimal

Use of JSDoc to document code functions

What could be improved ?

For high structured projects, use of Object Oriented Programming might be preferable

Use of Reactive Programming might be preferable to maintain variables in a flow, saving the need of manually updating them
